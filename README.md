# HintBot

リアルタイム音声認識とAIによるヒント提示システム

## 概要

HintBotは、マイクからの音声をリアルタイムで文字起こしし、設定されたトピックやフレーズを検出してヒントを表示するWebアプリケーションです。営業トレーニングやプレゼンテーション支援など、リアルタイムでのガイダンスが必要なシーンで活用できます。

## 主要機能

- **リアルタイム音声認識**: マイク入力をリアルタイムで文字起こし
- **トピック判定**: 会話の文脈からトピックを自動判定
- **フレーズ検出**: 特定のキーワードを検出してヒント表示
- **ヒント表示**: 仮判定→確定の2段階表示
- **設定管理**: トリガー条件とヒントの編集・保存

## セットアップ

### 必要な環境変数

```bash
NUXT_PUBLIC_OPENAI_API_KEY=your_openai_api_key
```

### インストール

```bash
npm install
```

### 開発サーバー起動

```bash
npm run dev
```

### ビルド＆デプロイ

```bash
npm run deploy
```
